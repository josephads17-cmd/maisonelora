# Maison Élora — Landing Page Premium

Version corrigée avec les images générées intégrées dans le dossier du site.

## Comment ouvrir le site

1. Dézippe `luxury-watch-landing-v2.zip`.
2. Ouvre `index.html` depuis le dossier extrait.
3. Garde bien le dossier `assets/` à côté de `index.html`, sinon les images ne s’afficheront pas.

## Structure

```text
luxury-watch-landing-v2/
├── index.html
├── README.md
└── assets/
    ├── images/       # images optimisées WebP + JPG utilisées par la page
    ├── originals/    # images originales fournies/générées
    └── ASSET-MANIFEST.md
```

## Important

- Les visuels packaging `packaging-moodboard-*` sont inclus uniquement comme inspiration. Ils ne sont pas utilisés comme promesse de packaging réel.
- La section cadeau utilise les visuels `gift-ad-*`, plus prudents, sans coffret précis.
- Nom de marque provisoire : `Maison Élora`. Remplace-le dès que tu as ton vrai branding.
- Les avis sont des placeholders à remplacer par de vrais avis clients.
